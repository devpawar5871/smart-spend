# 💰 Smart Spend Dashboard

An interactive **personal expense tracker web app** built using Streamlit.  
It helps users track spending, analyze expenses, and visualize financial trends.

## 🚀 Features

- ➕ Add daily expenses
- 📂 Upload CSV data
- 💾 Save expense records
- 📊 Category-wise pie chart
- 📅 Monthly expense bar chart
- 📈 Real-time metrics (Total, Average, Transactions)
- 🧾 Transaction history table

---

## 🛠️ Tech Stack

- Python 🐍
- Streamlit 🎈
- Pandas 📊
- Matplotlib 📉

---

## 📊 Dashboard Features

### 📌 Expense Tracking
Add expenses with category, amount, and description.

### 📌 Visualization
- Category-wise spending (Pie Chart)
- Monthly spending trend (Bar Chart)

### 📌 Data Management
- Upload CSV file
- Save data locally
